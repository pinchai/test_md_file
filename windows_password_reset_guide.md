# 🔐 Windows Password Reset Guide

## 1. If You Are Logged In (Change Password)

### Method 1: Using Keyboard Shortcut
1. Press **Ctrl + Alt + Delete**
2. Click **Change a password**
3. Enter your current password
4. Enter and confirm your new password

### Method 2: Using Settings
1. Go to **Settings → Accounts → Sign-in options**
2. Under **Password**, click **Change**
3. Follow the instructions

---

## 2. If You Forgot Password (Microsoft Account)

If your account uses an email (e.g., Outlook, Gmail):

### Method 1: From Login Screen
1. Click **I forgot my password**
2. Verify via email or phone
3. Set a new password

### Method 2: Online Reset
- Visit: https://account.microsoft.com/password/reset
- Follow the recovery steps

---

## 3. If You Forgot Local Account Password

### Option 1: Security Questions
1. Click **Reset password** on login screen
2. Answer security questions
3. Create a new password

### Option 2: Another Admin Account
1. Log in with an admin account
2. Go to **Control Panel → User Accounts → Manage another account**
3. Select the user and reset password

---

## 4. If All Methods Fail (Reset Windows)

1. On login screen, hold **Shift + Restart**
2. Select:
   - **Troubleshoot → Reset this PC**
3. Choose:
   - **Keep my files** (recommended)
   - or **Remove everything**

---

## ⚠️ Important Notes
- Resetting password may cause loss of encrypted data
- Use strong passwords
- Consider PIN or biometric login for convenience

---

## 📌 Tip
Always keep a backup admin account or recovery option enabled to avoid lockout.
